## 🧠 Why I built this
I was tired of seeing AI used for simple text generation. I wanted to see if I could engineer a "Swarm" that could handle the entire lifecycle of an **OSINT (Open Source Intelligence)** mission. 

The goal: Give the system a high-level directive (e.g., "Map all airports in a specific region"), and let the agents coordinate, extract coordinates, calculate distances, and build a relational map without a human touching the keyboard.

---

## 🛠️ The Technical Flex
This repository represents **28,000+ Lines of Code** of custom logic. It is part of a larger ecosystem totaling over 80,000 LOC These systems I build All alone from scratch.

### The Architecture
*   **The Kernel:** A Python/FastAPI backend that manages the "Heartbeat" of the agents and handles physical sandboxing for script execution.
*   **The Operational Theater:** A React-based UI that uses an off-thread **Spatial Physics Engine**. Even with 1,000+ nodes, the UI stays at a smooth 60FPS.
*   **The Geospatial Engine:** I integrated WebGL (MapLibre) to project raw extracted data onto a 3D globe with custom parabolic data arcs.
*   **The Logic Router:** Implements a custom **Visual Notation Markup (VNM)**. The agents talk in plain text, and my parser translates that text into physical nodes and edges on the map in real-time.

---

## 📡 Core Capabilities
*   **Autonomous Swarm Coordination:** Agents pass the "Pulse" to each other based on mission status.
*   **Semantic Deduplication:** The backend intelligently merges duplicate data points to keep the map clean.
*   **3D Tactical Visualization:** Real-time transition from 2D graphs to 3D physical coordinates.
*   **Persistent Blackboard:** A shared project memory that allows agents to remember what they’ve already researched.
